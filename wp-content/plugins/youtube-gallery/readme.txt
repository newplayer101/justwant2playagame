== Description ==
hkYouTubeGallary  is a WordPress plugin which is  developed by HuskerInfotech that allows you to manage and display Your Youtube Videos with all or single category on your wordpress blog.

This is without doubt the best YouTube Video plugin for WordPress, it is simple but powerful and flexible.

= ADD AND EDIT VIDEO EASILY =

To add a new Video, just enter the Name, Description,Thumb image in case you want to use lightbox and Youtube video url.
select category from right side.
create page if you want to display all vedio then add short code like,[hk_youtube] else if you want single category video then enter shortcode like [hk_youtube category="Your category Slug"].
For video settings, select settings option form menu.


== Installation ==

1. Upload hkYouTubeGallary folder to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Now you are done. Enjoy the plugin!!!
